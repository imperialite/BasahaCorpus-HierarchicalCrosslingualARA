Mga Gamit sa Pagpanguma


Othong KhamInsou




Ounla Santhi





Tion na para magtanom sang humay​





Kag Ang akon amay nagamit sang arado para aradohon ang amon taramnan





Gin usar man ni tatay ang labay para labayon ang dalayrikan





Gin gamit man ni tatay ang sadol para hagbason ang hilamon sakahon





Si manong naman ya nagamit sang pala agod hagbasan ang mga hilamon kahon





Si manang naman ya nag osar sang pisaw para kanalan ang talamnan





Gn gamit man ninanay ang binangon para utdon ang punta sang palay





Gin gamit ya man ang binangon para hagbasan ang mga hilamon sa kahon





Kag Sa tion sang tig alani tanan nag gamit sang garab para mag garab
