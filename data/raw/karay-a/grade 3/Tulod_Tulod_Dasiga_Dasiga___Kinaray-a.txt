Tulod Tulod Dasiga Dasiga





Nagsakay si uncle Lou sa ana nga sarakyan, garing wara na naghalin. Nag agto lang sa trrrr.. trrrrr…

May nagbulig bala ka tulod kang sarakyan?





Nag abot si Tiyo Cole para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Pira ang rudya?” Sara lang.

May nagbulig bala ka tulod kang sarakyan?&nbsp;





Nag abot si Tiyo Sam para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May darwa.&nbsp;





Nag abot si Tiyo Zed para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May tatlo.&nbsp;





Nag abot ang kabayo ni Tiyo Henry para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May apat.&nbsp;





Nag abot ang gamay nga bata para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May lima.&nbsp;





Nag abot ang tatay ni Hailey para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May anum.&nbsp;





Nag abot ang gamay gamay nga kuti para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May pito.&nbsp;





Nag abot ang Asno para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May walo.&nbsp;





Nag abot ang tatay ni Eden para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May siyam.&nbsp;





Nag abot ang bataon nga tawo para magbulig. Gintulod na kag gintulod.

“Dali, dasiga, dasiga, dasiga,” hambal ni tiyo Lou.

“Sa kadya, Pira run ang rudya?” May pulo.&nbsp;





Ang napulo ka tawo nagatulod kang sarakyan, kag wara man giyapon nagapanaw. Anhaw?

Ginsikadan ni Tiyo Lou ang preno.&nbsp;





Ang katapusan.
