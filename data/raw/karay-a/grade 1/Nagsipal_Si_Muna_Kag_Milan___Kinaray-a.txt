Nagsipal Si Muna Kag Milan





Nag-uli sa balay sanday Muna kag Milan halin sa zoo.





Amo run ako kadya.

Bao run ako kadya.





Elepante ako run kadya.





Lophophorus&nbsp;run ako kadya.

Rhino run ako kadya.





Kuneho run ako kadya.

Tigre run ako kadya.





Oras dun para magturog. Ginpatay ni nanay ang sulo kag naghalin. Ginmukra ni Muna ng mga mata kag naghambal "Sa kadya, bukaw run bala ako."
