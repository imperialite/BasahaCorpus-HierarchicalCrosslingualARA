Kaya kag Indi Kaya





Indi ako kahibalo maggantsilyo, ang mga kuring indi kasulat, ang mga isda indi kapamisok...





... Ang mga baboy indi kalupad!





Ang mga ido indi ka-saot, ang mga subay indi kabasa, ang mga uwak indi ka-kanta...&nbsp;





... ang mga amo indi ka-luto!&nbsp;





Apang, makahabyog ang mga amo!





Makahuni ang mga uwak.... maka-kutot ang mga subay... maka-kagat ang mga ido!





Makakaon ang mga babaoy, makalumpat ang mga kuring... kag ako makabasa!
