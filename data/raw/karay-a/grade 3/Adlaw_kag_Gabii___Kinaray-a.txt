Adlaw kag Gabii





Kang una, tama gid ka tamad ang adlaw. Naga butlak lang tana kun gusto na.





Naga tukturuok run ang sulog, pero wara gihapon nagabutlak ang adlaw.





Indi tana magbutlak ang adlaw. Kundi nga ang bituon kag ang bulan naga bira-bira gid ka obra nga wara ti pahuway.





Ginagtunan nanda ang adlaw para hurayun ang oras pero wara tana gasapak ang adlaw.





Nangayo kang bulig ang bulan kag ang bituon sa langit.





Ginhuray kang langit ang adlaw kag ang gabii kananda.





Kun indi magpati ang adlaw, indi run tana pagsugtan maagto sa langit.





Ang sulog naga panukturuok kun magbutlak ang adlaw sa kasanagun kag magsalup sa sirum.





Ang bulan kag ang mga bituon naga siga kun gabii.





Halin kato, matutum run gid ang adlaw nga nagaobra kag naga siga kun adlawun.





Ang adlaw, ang bulan, kag ang mga bituon masinadyahun nga nagabutlak sa insakto nga oras.
