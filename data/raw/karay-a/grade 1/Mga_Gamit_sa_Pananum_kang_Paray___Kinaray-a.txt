Mga Gamit sa Pananum kang Paray





Oras run para mananum kang paray.





Si tatay nagagamit kang singkaw kag arado para araduhon ang taramnam.





Si tatay nagagamit kang suyud para suyudon ang taramnam.





Si tatay nagagamit kang sadol pangsadol kang lupa para kahunan ang taramnan.





Si manong nagagamit kang pala para palahon ang lupa ibutang sa kahon kang taramnan.&nbsp;





Si manang nagagamit kang taratara pamukpuk kang lupa para kahunan ang taramnan.





Si nanay nagagamit kang garab pang-utod oway kang paray.





Si nanay nagagamit kang espading para hawanan ang hilamon sa kahon kang taramnan.





Ang tanan nagagamit kang garab nga iririmaw mangani.&nbsp;
