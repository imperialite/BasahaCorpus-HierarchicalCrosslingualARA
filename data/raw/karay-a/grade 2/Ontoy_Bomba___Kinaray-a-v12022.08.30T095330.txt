Ontoy Bomba


Prabha Bhattarai




Ujjwal Moktan





Bugtaw run si Ontoy Bomba!

Duro ang nagakinahanglan kang tubig nga ana ginapailig.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Lola.

"Sige, isalod lang ang imo nga pitsel rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Puwede run ako makaparigos kay masimba pa ako," sabat ni Lola.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Lita.

"Sige, isalod lang ang imo nga banga rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Puwede run ako makaraha ka amun pamahaw," sabat ni Lita.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Tatay Lito.

"Sige, isalod lang ang imo nga balde rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Mahugasan ko run ang akun motor," sabat ni Tatay Lito.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Lota.

"Sige, isalod lang ang imo nga labador rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Makapanglaba run ako ka akun mga bayo," sabat ni Lota.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Liling.

"Sige, isalod lang ang imo nga tasa rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Puwede ko run mahugasan ang akun nga paint brush," sabat ni Liling.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Luna.

"Sige, isalod lang ang imo nga sprinkler rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Puwede ko run mabunyagan ang akun mga tanum nga laswa," sabat ni Luna.





"Ontoy Bomba, puwede mo ako matugruan kang tubig?" pamangkot ni Lino.

"Sige, isalod lang ang imo nga balde rugya," hambal ni Ontoy Bomba.

"Salamat, Ontoy Bomba. Puwede run ako ka parigos imaw ka akun mga barkada," sabat ni Lino.





"Ontoy Bomba, puwede mo ako tugruan kang tubig?" pamangkot ka ayam.

"Sige, itangub lang rugya. Inum para ang imo nga uhaw madura," hambal ni Ontoy Bomba.





"Salamat kag natapos run ang adlaw. Maturog run ako. Sa rum-an, duro liwat ang mangayo ka akun tubig," hambal ni Ontoy Bomba.





Aga run.

Handa liwat si Ontoy Bomba sa ana nga mga orubrahon.
